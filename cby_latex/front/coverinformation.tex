% 中文摘要
\begin{cabstract}

蛋白质功能预测是生物信息学领域的核心任务之一。随着高通量测序技术的飞速发展，蛋白质序列数据呈指数级增长，然而经实验验证的功能注释比例极低，序列-功能鸿沟日益加深。现有的深度学习方法通常仅依赖序列模式特征，将蛋白质域信息和基因本体（Gene Ontology, GO）功能标签视为简单的分类标识符，忽略了蕴含于文本定义中的丰富语义信息，从而限制了模型对未见标签的泛化能力。针对上述问题，本文围绕"多模态语义融合"与"预训练域感知适配"两个方向展开研究，提出了两个具有递进关系的蛋白质功能预测框架。

首先，提出多模态零样本蛋白质功能注释方法MZSGO。该方法将蛋白质功能预测重新定义为蛋白质-GO匹配问题，利用蛋白质语言模型ESM2提取序列进化特征，利用大语言模型Qwen3-Embedding对蛋白质域描述和GO功能定义进行统一语义编码，通过三路特征投影将异构表示映射至共享语义空间。设计了自适应门控融合模块动态调节各模态贡献权重，并引入非对称特征丢弃策略增强模型对模态缺失的鲁棒性。实验表明，MZSGO在标准测试集上的有监督指标达到最优水平，尤其在零样本场景下显著超越现有方法。

其次，为进一步提升序列表征的功能感知能力，提出基于域感知适配器预训练的端到端蛋白质功能预测方法MZSGO-DA。该方法设计了轻量级适配器模块，以即插即用方式注入蛋白质语言模型的Transformer层内部；通过对比学习预训练，将蛋白质域的语义信息蒸馏至序列编码器，使模型仅凭序列输入即可产出携带域信息的嵌入表示。在下游阶段，采用双适配器策略实现端到端训练，兼顾预训练知识保留与任务特定适配。实验结果表明，MZSGO-DA在端到端场景下进一步提升了功能预测性能。

本文的创新性体现在：一方面，通过引入大语言模型构建域和标签的统一语义空间，实现了蛋白质功能预测从封闭标签集到开放标签集的范式转换，使模型具备零样本推理能力；另一方面，通过域感知对比学习预训练与双适配器端到端训练机制，在不改变蛋白质语言模型原始参数的前提下，高效地将多模态知识注入序列编码器，提升了模型的表征能力与泛化性能。


\end{cabstract}

% 英文摘要
\begin{eabstract}

Protein function prediction is a fundamental task in bioinformatics. With the rapid advancement of high-throughput sequencing technologies, protein sequence data has grown exponentially, yet the proportion of experimentally validated functional annotations remains extremely low, deepening the sequence-function gap. Existing deep learning methods typically rely solely on sequence pattern features, treating protein domain information and Gene Ontology (GO) functional labels as simple categorical identifiers while neglecting the rich semantic information embedded in their textual definitions. This limits model generalization to unseen labels. To address these challenges, this thesis investigates two complementary directions—multimodal semantic fusion and pretrained domain-aware adaptation—and proposes two progressively advanced protein function prediction frameworks.

First, we propose MZSGO, a Multimodal Zero-Shot protein function annotation method. MZSGO redefines protein function prediction as a protein-GO matching problem. It employs the protein language model ESM2 to extract evolutionary sequence features and utilizes the large language model Qwen3-Embedding as a unified semantic encoder for both protein domain descriptions and GO functional definitions. Three independent projection modules map these heterogeneous representations into a shared latent space. An adaptive gated fusion module dynamically adjusts the contribution weights of each modality, while an asymmetric feature dropout strategy enhances robustness against modality absence. Experiments demonstrate that MZSGO achieves state-of-the-art performance on standard benchmarks, with particularly significant improvements in zero-shot scenarios.

Second, to further enhance the functional awareness of sequence representations, we propose MZSGO-DA, a Domain-Aware adapter pretraining approach for end-to-end protein function prediction. This method designs lightweight adapter modules that are injected directly into the Transformer layers of the protein language model in a plug-and-play manner. Through contrastive learning pretraining, domain semantic information is distilled into the sequence encoder, enabling the model to produce domain-aware embeddings from sequence input alone. In the downstream stage, a dual-adapter strategy supports end-to-end training while preserving pretrained knowledge and enabling task-specific adaptation. Experimental results show that MZSGO-DA further improves prediction performance in the end-to-end setting.

The main contributions of this thesis are twofold: on one hand, by introducing large language models to construct a unified semantic space for domains and labels, we achieve a paradigm shift from closed-set to open-set protein function prediction, endowing the model with zero-shot reasoning capabilities; on the other hand, through domain-aware contrastive pretraining and a dual-adapter end-to-end training mechanism, we efficiently inject multimodal knowledge into the sequence encoder without modifying the original protein language model parameters, enhancing both representational capacity and generalization performance.


\end{eabstract}
